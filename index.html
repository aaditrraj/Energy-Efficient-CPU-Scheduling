How EATS Works

DVFS Control

Dynamically adjusts CPU voltage and frequency. Power scales as P = C·V²·f, so reducing voltage by 40% cuts power by ~64%.

Thermal-Aware

Monitors die temperature using an RC thermal model. Soft-throttles at 75°C and forces minimum frequency at 90°C.

Priority Scheduling

Critical tasks get highest priority with EDF ordering. Background tasks run at minimum power when no deadlines are urgent.

Core Management

Activates only the cores needed for current workload, reducing static/leakage power by up to 75% when idle.
